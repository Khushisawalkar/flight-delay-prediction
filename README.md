
# ✈️ Flight Delay Prediction System

This project predicts whether a flight will be delayed using machine learning techniques on historical flight data.

---

## 📊 Problem Statement
Flight delays impact passengers and airline operations. The goal of this project is to analyze flight data and build a model that can predict delays based on key factors.

## 🛠️ Tech Stack

- Scikit-learn  
- Matplotlib  
- Streamlit

## ⚙️ Approach

- Feature selection and basic feature engineering
